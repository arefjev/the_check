Подключение
Для получения доступа к API-интерфейсу сервиса Пользователю необходимо направить заявку
на подключение с указанием следующих сведений:
- aдрес Интернет-площадки или название Веб-приложения;
- тематику Приложения Пользователя;
- идентифицирующие Пользователя данные;
- адрес электронной почты Пользователя для отправки Ключа доступа.

Статистика запросов
Для получения статистики Запросов Пользователя к API-интерфейсу необходимо выполнить обращение по адресу:
https://proverkacheka.com/check/statistics?token=:token
где:
:token – Токен доступа
Отобразиться таблица с информацией:
Дата (год/месяц)
Кол-во успешных обращений
Кол-во неуспешных обращений

Формат запроса
POST https://proverkacheka.com/api/v1/check/get
Тело запроса:
fn - Номер фискального накопителя
fd - Номер фискального документа
fp - Фискальный признак документа
t - Дата и время
n - Тип операции (Приход/Возврат прихода/Расход/Возврат расхода)
s - Сумма чека
qr – Признак сканирования qr кода (0/1)
token – Токен доступа
